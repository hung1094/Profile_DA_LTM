Java không chỉ là một ngôn ngữ lập trình, nó là nền tảng cho sự phát triển của internet hiện đại. Ra đời năm 1995 với triết lý **"Write Once, Run Anywhere"** (WORA), Java đã thay đổi cách chúng ta xây dựng các ứng dụng phân tán.

### Tại sao chọn Java cho lập trình mạng?

Trong lập trình mạng, Java sở hữu những ưu điểm vượt trội mà không phải ngôn ngữ nào cũng có được:

1.  **Thư viện chuẩn mạnh mẽ:** Gói `java.net` cung cấp các API bậc cao giúp lập trình viên không cần quan tâm đến các chi tiết phức tạp của tầng vận chuyển.
2.  **Quản lý bộ nhớ tự động:** Cơ chế _Garbage Collection_ giúp giảm thiểu lỗi rò rỉ bộ nhớ (memory leak) – nỗi ác mộng của các server hoạt động 24/7.
3.  **Bảo mật tuyệt đối:** Java có cơ chế _Security Manager_ và kiểm soát truy cập mạnh mẽ, điều cốt lõi cho các giao dịch ngân hàng và doanh nghiệp.

> **Sự thật thú vị:** Hầu hết các hệ thống giao dịch chứng khoán và lõi ngân hàng hiện nay vẫn chạy trên JVM nhờ sự ổn định và hiệu năng xử lý luồng (stream) cực kỳ tốt.
